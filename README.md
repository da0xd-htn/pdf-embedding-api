# PDF Embedding/Extraction API  
## Cloud Computing Mini-Project  
**Description**: Flask API to embed/extract PDFs with Swagger docs.  

### Technologies  
- Python 3.9  
- Flask  
- PyPDF2  
- Docker  
- Azure

git clone https://github.com/yourusername/pdf-embedding-api.git
